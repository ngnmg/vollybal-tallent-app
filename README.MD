# Volleyball Talent App

یک اپلیکیشن ساده استعدادیابی والیبال با سه مرحله:
- مرحله ۱: اطلاعات متنی بازیکن و پیشنهاد پوزیشن.
- مرحله ۲: آپلود تصویر/ویدئو (نسخه نمایشی تحلیل حرکتی).
- مرحله ۳: پیش‌بینی نتیجه مسابقه والیبال با مدل ساده.
